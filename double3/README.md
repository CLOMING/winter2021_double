# App
OpenCV based app
## Installiation
### Python `3.10.0`
```bash
pip install -r requirements.txt
```
### Opencv
We need to build opencv manually to use Nvidia Jetson TX2
See [install-opencv.md](install-opencv.md).
## Run
```bash
bash run.sh
```
